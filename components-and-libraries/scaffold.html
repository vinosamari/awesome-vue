# Scaffold

Scaffold / boilerplate / seed / starter kits / stack ensemble / Yeoman generator

  • vue-cli (opens new window) - Simple CLI for scaffolding Vue.js projects.
  • Vue-Django (opens new window) - A boilerplate to set you up in bringing the awesomeness of VueJS into a Django (Python) app.
  • python-vuejs (opens new window) - Gluing Python web frameworks and Vue.js with a set of scripts. Basically a vue-cli wrapper.
  • generator-vue-plugin (opens new window) - Yeoman generator generating vue plugin.
  • vue-seed (opens new window) - vue-seed is minimal seed for those looking to get up-and-running with Vue
  • nuxt-seed (opens new window) - nuxt-seed is minimal seed for those looking to get up-and-running with Vue and Nuxt
  • rails_vue_melt (opens new window) - Rails view with webpack=vue optimizer.
  • vue-starter (opens new window) - A Vue.js starter kit that lets you focus on more programming and less configuration.
  • vuejs-wordpress-theme-starter (opens new window) - A WordPress theme with the guts ripped out and replaced with Vue
  • Cordovue (opens new window) - A sample Apache Cordova application using Vue.
  • Cookiecutter-Django-Vue (opens new window) - Django+VueJS+Docker customizable project generator with a large number of settings/integrations
  • iBiu (opens new window) - A visual CLI for scaffolding large Vue projects in 2 seconds.
  • wp-vue (opens new window) - A simple Vue blog template that displays posts from a WordPress REST API endpoint.
  • vue-cli-template-nativescript (opens new window) - Template for starting new nativescript+vue projects with rollup+babel+eslint
  • vue-element-ui-scaffold (opens new window) - Online and visual Vue 2 with Element-UI CRUD scaffold/generator.
  • vue-firebase-element-ui-scaffold (opens new window) - Online and visual Vue 2, Element-UI and Firebase scaffold/generator with CRUDs, authentication, file/image upload, activity log and more.
  • vuesion (opens new window) - The most complete boilerplate for production-ready PWAs. With focus on performance, development speed, and best practices
  • vue-vuex-typescript-webpack-seed (opens new window) - A seed project with Vue, Vuex, Typescript, Scss & Webpack with hot reloading
  • VueCharged Template (opens new window) - A highly opinionated, feature-first Vue 2 template with CLI scaffolding. Uses Vue 2, Vuex, Vue Router and Vue I18n.
  • Huncwot (opens new window) - Vue.js boilerplate with Node.js for building modern JavaScript applications with « batteries included » approach.
  • wemake-vue-template (opens new window) - Bleeding edge vue template focused on code quality and developer happiness. Featuring: nuxt, flow, and jest.
  • vue-lib-template (opens new window) - A simple template for building and publishing Vue component/library as a open source project
  • Awesome Vue Boilerplate (opens new window) - Awesome Vue, Vuex, Vuex-pathify, element-ui, tailwindcss
  • ScaffoldHub.io (opens new window) - Generate full Vue applications with SQL, MongoDB or Firebase Firestore databases.
  • VuePlay (opens new window) - Generate disposable Vue playgrounds in seconds. Allows you to test things quickly.
  • Mevn-CLI (opens new window) - Light speed setup for MEVN stack based apps.
  • vue-cli-template-registry (opens new window) - A solution for installing vue-cli custom templates hosted on private/enterprise repositories.
  • Vuejs Firebase CRUD Starter with Auth (opens new window)
  • vue-enterprise-boilerplate (opens new window) - An ever-evolving, very opinionated architecture and dev environment for new Vue SPA projects using Vue CLI 3
  • nuxt-headless (opens new window) - Boilerplate for Nuxt.js using Wordpress REST API as headless CMS
  • vue-starters-directory (opens new window) - Search for available scaffold projects and starter kits for VueJS. Features search and github stats are available
  • janak (opens new window) - Build your Vue 3 apps in a breeze
  • vue-composable-starter (opens new window) - Minimalist starting point for your next Vue composable
  • Vuetify-tailwind-animate-starter (opens new window) - A vue-cli starter template for Vuetify + Tailwind CSS + Animate.css
  • vuejs-vuetify-structured-template boilerplate (opens new window) - A vue-cli template for a full-featured Vue.js + Vuetify + great structure + API ready

# Client

Render Vue application in the browser only

  • Bourgeon (opens new window) - Bourgeon is an opinionated-featured VueJS 2.0 setup for Webpack.
  • Vue Settler (opens new window) - An opinionated Vue 2.0 SPA Starter.
  • vue-multiple-pages (opens new window) - A modern Vue.js multiple pages starter which uses Vue 2, Webpack2, and Element-UI
  • vue-typescript-boilerplate (opens new window) - A Vue.js typescript SPA starter with Vue 2, Vue Typed, Vuex, Vue Router and localization
  • vue-tachyons-template (opens new window) - A Vue 2 project starter template w/ Tachyons, Webpack, and ESLint
  • Vuets (opens new window) - A Vue, TypeScript ready boilerplate using class-style components, vue plugin options, webpack & vue-cli.
  • MMF-FE/vue-typescript (opens new window) - A vue2.x typescript template.
  • Friendly Vue Starter (opens new window) - A full-featured Vue.js starter project with GraphQL support via Apollo-client (Vuex, Vue-router, Vue-i18n, Webpack 3, Eslint, Prettier, ...)
  • vue-ts-amd (opens new window) - A full-featured Vue.js 2 boilerplate using AMD pattern (RequireJS) and Typescript.
  • vue-2-boilerplate (opens new window) - Vue 2 boilerplate for developing medium to large single page applications by petervmeijgaard (opens new window)
  • vue-cli-template-library (opens new window) - Template for developing open-source vue.js libraries with Rollup + Jest + Babel + Storybook + TravisCI + SemanticRelease.
  • vue-cli-template-github-pages (opens new window) - A full-featured Webpack + vue-loader setup for Github Pages Deployment with travisCI.
  • vue-webpack-chrome-extension-template (opens new window) - Template for quick creation of Chrome extension on Vuejs hot reloading when developing.
  • vue-auth-boilerplate (opens new window) - Vue SPA boilerplate with Router/Vuex/CLI3 and auth functions (cool looking register and login). Works with minimal setup out of the box with laravel-api-boilerplate-jwt (opens new window).
  • vue-atomic-design (opens new window) - Vue front-end boilerplate based on atomic design methodology.
  • vue3-compact-template (opens new window) - A simple and compact Vue 3 template with current cutting edge front-end technologies

# Universal

Render Vue application to HTML on the server and to the DOM in the browser

  • SPA Starter Kit (opens new window) - A highly opinionated starter kit for building Single Page Applications with Laravel and Vue.js.
  • SSR Boilerplate (opens new window) - Vue.js server-side rendering boilerplate without polluting Vuex
  • neutrino-preset-vue-static (opens new window) - A minimalistic starter kit for building static sites using Vue.js.
  • Vueniverse (opens new window) - A fully featured, universal Vue template for user-based applications, powered by Nuxt.js and Express.
  • vue-preload (opens new window) - A plugin Vue for adding <link rel="preload" capabilities
  • Vue.js/Nuxt.js full-featured boilerplate (opens new window) - Professional agency front-end template for building fast, robust, and adaptable web apps or sites by Vue.js/Nuxt.js

# Server

Includes a backend boilerplate

  • Vuetober (opens new window) - Single page apps with Laravel, Vue.js, and October CMS.
  • vue-server-demo (opens new window) - Writing koa server in Vue.
  • vue-apollo-connector-kit (opens new window) - JWT authentication with 2 different strategies (localStorage and httpOnly cookies). Authorization using custom directives (apollo).
  • DotVue (opens new window) - Implement .vue file handler in .NET with server ViewModel postback. Single-file components with server-side ViewModel. Use all power of Vue.js with simple C# server-side data access
  • isomor-vue-app (opens new window) - Setup a working environment with Vue and isomor, to unify the frontend and the backend by generating automatically the API.

# Electron

  • Electron Vue (opens new window) - An Electron & Vue.js quick start boilerplate with vue-cli scaffolding, common Vue plugins, electron-packager/electron-builder, unit/e2e testing, vue-devtools, and webpack.
  • VuePack (opens new window) - A modern starter which uses Vue 2, Vuex, Vue-router and Webpack 2 (and even Electron).
  • vue-flexible-link (opens new window) - Tiny Vue component for Electron to open links in a browser. Ideal for cross-environment apps (web & native)
  • vuelectron (opens new window) - electronjs starter kits for vue.
  • vue-design (opens new window) - the best website visualization builder with Vue and Electron.
  • dapp (opens new window) - Boilerplate for stack of Vue/Parcel/Electron.
  • vue-electron-template (opens new window) - An Electron & Vue.js boilerplate with Hot-reloading enabled and common Vue plugins, dev and build scripts configured using webpack-4
  • vue-iview-electron-boilerplate (opens new window) - An Electron & Vue.js boilerplate with iview 4.x, Vue 2.x, Electron 8.x and Webpack 4.x
  • electron-vue3-flask (opens new window) - An Electron & Vue.js application starter template that uses a Python backend via Flask for desktop application building

# Parts

Scaffold parts of the app

  • vue-generator (opens new window) - A CLI generator for Vue components, views and store modules.
  • vue-component-generator (opens new window) - A generator for vue component, supports Vue1.X and Vue2.X.
  • rapydml_cmp (opens new window) - Vue-component pythonic generator, built on top of RapydML & RapydScript.
  • vue-authoring-template (opens new window) - Template for authoring also publishing Vue component and their use case in story-scenario (storybook) format in easy way.
  • vue-component-boilerplate (opens new window) - Template for generating the boilerplate code for a Vue.js component, with build scripts and testing pre-installed
  • vue-generate-component-typescript (opens new window) - A typescript code generator for vue, it can generate components, directives, filters.
  • vuejs-generate (opens new window) - An easy CLI for generating Vue components, stores, store modules & unit tests
  • vue-sh-cli (opens new window) - A CLI for generating Vue components, views, store modules & unit tests. Supports Vue 3 components template and TypeScript
