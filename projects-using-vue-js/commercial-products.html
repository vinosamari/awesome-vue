# Commercial Products

  • Wijmo (opens new window) - A collection of UI controls with VueJS support.
  • ChatWoot (opens new window) - Livechat and agent collaboration over Facebook messenger.
  • VueA (opens new window) - VueJS Admin template with multiple layouts and laravel version.
  • Teleo (opens new window) - Team collab-app moving effortlessly between talking, planning & doing
  • EducationLink (opens new window) - CRM and sales automation for education agents and colleges.
  • Pragmatic v2.0 (opens new window) - Responsive and configurable admin template built with Vue.js and Element.
  • Moonitor (opens new window) - Cryptocurrency tracker for Desktop.
  • Deskree (opens new window) - Online collaboration platform that combines Ideas, Tasks, and Issues in one place.
  • OSHCExpress (opens new window) - A comparison and ecommerce for OSHC (Overseas Student Health Cover) insurance (Australia's insurance for international students).
  • Agiloo (opens new window) - Project Management app for Scrum and Kanban
  • ScaffoldHub (opens new window) - Online Web App Generator for VueJS with NodeJS, and MongoDB or SQL.
  • Commandeer (opens new window) - Cloud Management Reimagined. A Desktop cloud management app built with Vue.js and Electron.
  • Mongster (opens new window) - Connect your Mongo DB nodes into one cluster within a control panel
  • Leave Dates (opens new window) - A powerful new way to track your staff leave
  • Time Door (opens new window) - A time series analysis API
  • vREST NG (opens new window) - An enterprise application for Automated API Testing, built with VueJS and Element UI
  • ScaleChamp (opens new window) - Multi-cloud managed databases provider with Hetzner, AWS, Linode, IBM, Azure, Scaleway, Alibaba Cloud, DigitalOcean, GCP and UpCloud support
  • Coloban (opens new window) - All-in-one project management tool with chats, Kanban, Gantt, calls, screenshare and many more
  • NxShell (opens new window) - An easy to use new terminal for SSH, which is based on Electron and Vue.js
  • Materio Vuetify VueJS Admin Template (opens new window) - Most powerful, developer-friendly, production ready & comprehensive Vuetify VueJS admin template
  • Mystery Football Shirts (opens new window) - An E-commerce app using Vue, Node & Mongdo DB
  • NocoDB (opens new window) - An opensource Airtable alternative
  • KodaDot (opens new window) - NFT Marketplace on Polkadot funded as public good, written in Vue.js
